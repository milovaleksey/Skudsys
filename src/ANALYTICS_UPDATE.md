## 🚀 Использование фильтров:

### Фильтр по годам:
```
Год от: 2024
Год до: 2025
```
→ Показывает данные только за 2024-2025 годы

### Фильтр по корпусам:
```
Корпус: Корпус №5
```
→ Показывает только проходы через Корпус №5

### Комбинация фильтров:
```
Год от: 2024
Год до: 2025
Корпус: Общежитие №1
```
→ Показывает проходы через Общежитие №1 за 2024-2025
