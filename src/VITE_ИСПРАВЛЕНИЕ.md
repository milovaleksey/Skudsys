## ❓ Частые вопросы

### 1. Какую команду использовать на production сервере?

**Ответ:** Из корня проекта (как обычно):
```bash
npm run build
```

### 2. Почему у меня две структуры (корень и /frontend)?

**Ответ:** Это monorepo подход. Но компоненты находятся только в корне. Исправления настроили `/frontend/` чтобы он правильно импортировал компоненты из корня.

### 3. Нужно ли что-то менять на production сервере?

**Ответ:** НЕТ! Если там работает - не трогайте. Исправления нужны только для локальной разработки.

### 4. Почему возникает ошибка "Cannot find module ./components"?

**Ответ:** Потому что `/frontend/App.tsx` пытался импортировать из `./components/`, которой там нет. Теперь исправлено на `../components/`.

### 5. Можно ли запускать из /frontend после исправления?

**Ответ:** ДА! Теперь можно запускать откуда угодно.
