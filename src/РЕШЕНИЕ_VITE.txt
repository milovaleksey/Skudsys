      plugins: [],  // БЕЗ плагина!

        loader: 'tsx',
        include: /\.[jt]sx?$/,
      },
      optimizeDeps: {
        esbuildOptions: {
          loader: {
            '.js': 'jsx',
            '.ts': 'tsx',
            '.tsx': 'tsx',
          },
        },
      },
    });

  Vite 6 имеет ВСТРОЕННУЮ поддержку JSX/TSX через esbuild!
  
  Плагин @vitejs/plugin-react нужен только для:
    • Fast Refresh (горячая перезагрузка без потери состояния)
    • Специфичных React фич (Context, Hooks сохранение)
  
  Для базового React-приложения достаточно встроенного esbuild.
  
  Это работает быстрее и без зависимостей!
