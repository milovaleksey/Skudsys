# Инструкция по установке логотипа

## Как установить логотип ТюмГУ:

1. Переименуйте ваш файл логотипа в `logo.png`
2. Скопируйте файл `logo.png` в папку `/public/`
3. Убедитесь, что путь выглядит так: `/public/logo.png`

## Рекомендуемые параметры:
- Формат: PNG с прозрачным фоном
- Размер: 200x200px или 300x100px  
- Цвета: Синий #00aeef (фирменный цвет ТюмГУ)

## Где отображается:
- Страница входа (96px высота)
- Боковое меню - открытое (64px высота)
- Боковое меню - свёрнутое (48px высота)

## Команда для копирования (на сервере):
```bash
cp /путь/к/вашему/логотипу.png /root/utmn-security/public/logo.png
```

## Проверка:
После копирования обновите страницу в браузере (Ctrl + R)
