# ⚡ Быстрый старт - 3 шага!

## 🔴 Важно! После изменения `.env` файлов:

Backend **ОБЯЗАТЕЛЬНО** нужно перезапустить! Node.js читает `.env` только при старте!

---

## 🚀 Три простых шага:

### 1️⃣ Остановите старый backend
```bash
# В терминале где запущен backend нажмите:
Ctrl+C

# Или убейте процесс:
killall -9 node
```

---

### 2️⃣ Запустите backend заново
```bash
cd /var/www/utmn-security/debug/backend
npm start
```

**Проверьте логи! Должно быть:**
```
[MQTT] Подключение к брокеру: mqtt://10.101.221.198:1883  ← ПРАВИЛЬНО!
[Parking MQTT] Подключение к брокеру: mqtt://10.101.221.198:1883  ← ПРАВИЛЬНО!
```

**НЕ должно быть:**
```
[MQTT] Подключение к брокеру: mqtt://localhost:1883  ← НЕПРАВИЛЬНО!
```

---

### 3️⃣ Обновите браузер
```
Ctrl + Shift + R  (жесткая перезагрузка)
```

**Проверьте консоль (F12):**
```
[WebSocket] ✅ Подключено  ← ХОРОШО!
[Parking WebSocket] Подключено  ← ХОРОШО!
```

**НЕ должно быть:**
```
429 (Too Many Requests)  ← ПЛОХО!
WebSocket connection failed  ← ПЛОХО!
```

---

## ✅ Готово!

Парковка должна работать! 🎉

---

## 🆘 Если не работает:

1. Проверьте, что backend **точно** перезапущен
2. Проверьте, что в логах `10.101.221.198` (не localhost)
3. Очистите кэш браузера (Ctrl+Shift+Delete)
4. Смотрите `/RESTART_INSTRUCTIONS.md` для детальной диагностики
