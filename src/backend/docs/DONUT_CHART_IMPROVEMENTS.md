# Улучшения диаграммы-пончика

## Проблема
Диаграмма-пончик не корректно отображала маленькие значения (менее 1-2%), и общее число в центре могло быть неправильным.

## Решение

### 1. Правильный подсчет общей суммы
```typescript
// Вычисляем общее количество из данных
const total = chartData.reduce((sum, item) => sum + item.value, 0);
```

Теперь число "Всего" в центре диаграммы **всегда правильное** - это сумма всех значений из данных.

### 2. Умные метки для сегментов
```typescript
const renderCustomLabel = (entry: any) => {
  const percent = (entry.value / total) * 100;
  
  // Показываем название + процент для сегментов >= 3%
  if (percent >= 3) {
    return `${entry.name} ${percent.toFixed(0)}%`;
  }
  
  // Только процент для сегментов 1-3%
  if (percent >= 1) {
    return `${percent.toFixed(0)}%`;
  }
  
  // Ничего не показываем для сегментов < 1%
  return '';
};
```

**Результат:**
- Большие сегменты (≥3%): "Китай 34%"
- Средние сегменты (1-3%): "2%"  
- Маленькие сегменты (<1%): без метки (но видны на диаграмме!)

### 3. Уменьшенный отступ между сегментами
```typescript
paddingAngle={1}  // Было 2, стало 1
```

Маленькие сегменты теперь лучше видны на диаграмме.

### 4. Легенда с точными данными
В легенде справа всегда показываются:
- Название страны
- Точное количество студентов
- Точный процент (с одним знаком после запятой)

**Пример:**
```
🔵 Китай         150    33.8%
🔵 Казахстан      89    20.0%
🔵 Грузия          1     0.2%
```

### 5. Абсолютное позиционирование центра
```typescript
<div className="absolute top-1/2 left-1/2 transform -translate-x-1/2 -translate-y-1/2">
  <div className="text-3xl font-bold">{total}</div>
  <div className="text-sm text-gray-500">Всего</div>
</div>
```

Число "Всего" теперь точно по центру диаграммы.

   - ✅ Метки не перекрываются
   - ✅ В легенде проценты в сумме дают 100%

## Возможности настройки

### Изменить порог отображения меток
В `DonutChartCard.tsx`:
```typescript
if (percent >= 3) {  // Измените это значение
  return `${entry.name} ${percent.toFixed(0)}%`;
}
```

### Изменить отступ между сегментами
```typescript
paddingAngle={1}  // 0 = без отступа, 5 = большой отступ
```

### Изменить размер диаграммы
```typescript
innerRadius={60}   // Внутренний радиус (дырка)
outerRadius={100}  // Внешний радиус (размер)
```

## Формула расчета

Все проценты считаются по формуле:
```
процент = (значение / общая_сумма) × 100
```

Где `общая_сумма` = сумма всех `students_count` из массива данных.
