# Обновление: Учет всех записей включая null

## Изменения

### Проблема
Пользователь хотел, чтобы ВСЕ записи учитывались в диаграмме, включая:
- `{"country": null, "students_count": 1}`
- `{"country": "Лицо без гражданства", "students_count": 9}`

Раньше компонент фильтровал записи с `country === null`.

### Решение

#### 1. Убрана строгая фильтрация
Теперь компонент `/components/DonutChartCard.tsx`:
- ✅ Принимает ВСЕ записи
- ✅ Автоматически заменяет `null` на "Без гражданства"
- ✅ Фильтрует только записи с `students_count === 0`

```typescript
// Заменяем null или пустую строку на "Без гражданства"
let countryName = item.country;
if (!countryName || countryName.trim() === '') {
  countryName = 'Без гражданства';
}
```

#### 2. Обновлен скрипт публикации
`/backend/scripts/publish-foreign-students-real.sh` теперь публикует:
- 65 записей (включая `country: null`)
- Сумма: **2360** студентов
- Все категории учитываются

## Объединение категорий (опционально)

Если нужно объединить "Лицо без гражданства" + null в одну категорию:

### Вариант 1: В базе данных
```sql
UPDATE students 
SET country = 'Лицо без гражданства' 
WHERE country IS NULL OR country = '';
```

### Вариант 2: В компоненте
```typescript
// В DonutChartCard.tsx
const chartData = data
  .map(item => ({
    name: !item.country || item.country.trim() === '' 
      ? 'Лицо без гражданства'  // Одно название для всех
      : item.country,
    value: Number(item.students_count) || 0
  }))
  .reduce((acc, curr) => {
    // Объединяем записи с одинаковым названием
    const existing = acc.find(x => x.name === curr.name);
    if (existing) {
      existing.value += curr.value;
    } else {
      acc.push(curr);
    }
    return acc;
  }, [] as Array<{name: string, value: number}>)
  .filter(item => item.value > 0);
```

Тогда будет:
- "Лицо без гражданства": 10 студентов (9 + 1)
- Всего: 65 категорий → 64 категории

## Что фильтруется

Компонент фильтрует ТОЛЬКО:
- ❌ `students_count === 0` - нулевые значения
- ✅ `country === null` - НЕ фильтруется, заменяется на "Без гражданства"
- ✅ `country === ""` - НЕ фильтруется, заменяется на "Без гражданства"
- ✅ Все остальное - отображается
