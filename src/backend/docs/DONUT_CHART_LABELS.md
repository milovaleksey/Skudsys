# Настройка меток диаграммы-пончика

## Логика отображения меток на диаграмме

В компоненте `/components/DonutChartCard.tsx` настроена умная система отображения меток:

### Правила отображения:

| Процент | Что показывается | Пример |
|---------|------------------|--------|
| **≥ 5%** | Название страны + процент | `КАЗАХСТАН 37%` |
| **2-5%** | Только процент | `3%` |
| **< 2%** | Нет метки | (пусто) |

### Код:

```typescript
const renderCustomLabel = (entry: any) => {
  const percent = (entry.value / total) * 100;
  
  // Показываем метку только если процент >= 5% (название + процент)
  if (percent >= 5) {
    return `${entry.name} ${percent.toFixed(0)}%`;
  }
  
  // Для средних значений (2-5%) показываем только процент
  if (percent >= 2) {
    return `${percent.toFixed(0)}%`;
  }
  
  // Для сегментов менее 2% не показываем метку вообще
  return '';
};
```

## Визуализация

```
        КАЗАХСТАН 37%  ← Большой сегмент (≥5%)
       ╱
    ╱─────╲
   │       │  3%  ← Средний сегмент (2-5%)
   │  2360 │
   │ Всего │      ← Маленькие сегменты (<2%) без меток
    ╲─────╱
       ╲
        ТУРКМЕНИЯ 33%
```

## Почему такая логика?

### ✅ Преимущества:

1. **Читаемость**
   - Не перегружается диаграмма мелкими метками
   - Крупные сегменты сразу видны
   - Мелкие не создают визуальный шум

2. **UX**
   - Важные данные (>5%) видны сразу
   - Средние данные (2-5%) помечены процентом
   - Мелкие данные (<2%) видны в легенде справа

3. **Производительность**
   - Меньше элементов на диаграмме
   - Быстрее рендеринг
   - Меньше наложений меток

### 📊 Для вашего случая:

Из 65 стран:
- **≥5%**: Казахстан (37%), Туркмения (33%), Таджикистан (7%), Узбекистан (5%)
- **2-5%**: Бангладеш (1.5%), Сирия (1.4%), и еще ~4 страны
- **<2%**: Остальные ~57 стран (видны только в легенде)

## Настройка порогов

Если нужно изменить пороги, отредактируйте функцию `renderCustomLabel`:

### Пример 1: Показывать только очень крупные (>10%)
```typescript
if (percent >= 10) {
  return `${entry.name} ${percent.toFixed(0)}%`;
}
return '';
```

### Пример 2: Показывать названия для всех >3%
```typescript
if (percent >= 3) {
  return `${entry.name} ${percent.toFixed(0)}%`;
}
if (percent >= 1) {
  return `${percent.toFixed(0)}%`;
}
return '';
```

### Пример 3: Вообще не показывать метки (только легенда)
```typescript
const renderCustomLabel = (entry: any) => {
  return ''; // Все метки скрыты
};
```

## Легенда

Легенда справа показывает **ВСЕ** записи, независимо от процента:

```
🔵 КАЗАХСТАН               875    37.1%
🔵 ТУРКМЕНИЯ               784    33.2%
🔵 ТАДЖИКИСТАН             162     6.9%
🔵 УЗБЕКИСТАН              122     5.2%
...
🔵 Лицо без гражданства      9     0.4%
🔵 Без гражданства           1     0.0%
```

- Цветной индикатор соответствует сегменту
- Название страны
- Количество студентов
- Точный процент (с одним знаком после запятой)

## Tooltip (всплывающая подсказка)

При наведении на ЛЮБОЙ сегмент показывается:

```
┌──────────────────────────┐
│ Количество               │
│ 875 студентов            │
└──────────────────────────┘
```

Работает даже для сегментов без меток!

## Рекомендации

### Для большого количества категорий (>50):
- ✅ Текущие настройки оптимальны
- ✅ Метки только для крупных сегментов (≥5%)
- ✅ Легенда со скроллом для всех

### Для среднего количества (10-30):
- Можно снизить порог до 3%:
  ```typescript
  if (percent >= 3) return `${entry.name} ${percent.toFixed(0)}%`;
  ```

### Для малого количества (<10):
- Можно показывать все метки:
  ```typescript
  return `${entry.name} ${percent.toFixed(0)}%`;
  ```

## Стилизация меток

Метки используют стили по умолчанию от Recharts. Для кастомизации:

```typescript
<Pie
  data={chartData}
  label={(entry) => ({
    content: renderCustomLabel(entry),
    style: {
      fontSize: '14px',
      fontWeight: 'bold',
      fill: '#1f2937' // gray-900
    }
  })}
/>
```

## Отключение меток вообще

Если нужно полностью убрать метки:

```typescript
<Pie
  data={chartData}
  // Удалить эти строки:
  // label={renderCustomLabel}
  // labelLine={false}
/>
```

Тогда диаграмма будет "чистой", без текста.
